# 📊 Meta Ad Performance Dashboard

🚀 **End-to-End Data Analytics Project using Power BI**

---

## 🔍 Overview

This project analyzes Meta (Facebook & Instagram) ad campaign performance and provides actionable insights to improve marketing ROI.

It covers the complete funnel:
👉 Awareness → Engagement → Conversion

---

## 🎯 Business Problem

Marketing teams struggle to:

* Track campaign performance
* Identify best-performing audience
* Optimize budget allocation

This dashboard solves these problems using data-driven insights.

---

## 📊 Key Metrics

* Impressions
* Clicks
* Engagements
* CTR (Click Through Rate)
* Conversion Rate
* Purchase Rate
* Total Budget

---

## 📈 Key Insights

✔️ High CTR (~11%) → Strong ad creatives
❌ Low Purchase Rate (~0.6%) → Weak conversion funnel
👩 Best Audience → Females (18–30)
🎥 Best Ad Type → Video & Stories
⏰ Best Time → Afternoon & Evening

---

## 🧠 Recommendations

* Improve landing page & conversion strategy
* Focus on high-performing audience segments
* Increase budget for Video & Story ads
* Run ads during peak engagement hours
